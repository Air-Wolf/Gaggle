It seems you haven't created any waypoints yet. There are two ways to add waypoints:

Manually
  • Just press the Menu button and choose Add waypoint
Importing WPT Files
  • Visit a waypoint (.wpt-only) file link on the web and choose Open with Gaggle
  • Many waypoint files can be found in the Worldwide Soaring Turnpoint Exchange
    • Choose either the CompeGPS or OziExplorer format
    • Try this example: Boulder, CO USA
  • If you'd like to see another format supported, please post a request in the user forum.
