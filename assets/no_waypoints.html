It seems you haven't created any waypoints yet. There are two ways to add waypoints:
  • If you visit a waypoint file on the web, when you open the file, just choose "Open with Gaggle". For an example, try downloading from the Norcal XC league or British Slovina Open.

    Currently only wpt format files are supported. If you'd like to see another format supported, please post a request in the user forum.


  • You can add waypoints manually. Just press Menu button and choose "Add waypoint."
